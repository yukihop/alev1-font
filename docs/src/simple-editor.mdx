## 簡易オンラインエディタ

入力欄にキーワードまたは文字コードを入れることで、対応するALEV文字に変換できます。

- **キーワード**は `love`, `perfect` のような英単語です。[文字一覧](/characters)を参照してください。すべて小文字で入力してください。
- **文字コード**は `0xFF` のような十六進文字列、あるいは `0b10110010` のような二進数表記です。
- キーワードと文字コードは混在していても構いません。半角スペースで区切ってください。
- <span style={{ color: "pink" }}>キーワードはALEV文字の解析が進むにつれて変更になることがあります</span>。その場限りの利用にとどめてください。将来にわたって安定して特定の字形で固定したい場合は、十六進または二進形式を使用して下さい。

<SimpleEditor defaultValue="i receive straylight 0b11001100 0xCC" />

文字はOSのスクリーンショット機能で保存してください。
