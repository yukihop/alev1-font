# Alevish &mdash; ALEV-1 FONT

<AlevSignalDemo />

Alevishは256個の概念グリフを持つ記号フォントです :love:

## メニュー

- 対応している文字および概念の一覧 → [文字一覧](/characters)
- 今すぐこのブラウザ上でALEV文字を打ち込みたい → [簡易オンラインエディタ](/simple-editor)
- ある程度複雑な日本語・英語・ALEV文字混じりの文書を作成したい → [Markdownエディタ](/editor)
- PowerPointやExcelなど、デスクトップでフォントを使いたい → [インストール](/install)
- 自分のウェブサイトにALEV文字を組み込みたい → [インストール](/install)
- 開発協力、語彙の拡張、辞書の修正 → [GitHub](https://github.com/yukihop/alev1-font) または作者の X へ
