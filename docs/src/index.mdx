ここは ALEV-1 は 256 個の概念グリフを持つフォント、ALEV-1 フォントのダウンロード兼デモサイトです :love:

- 対応している文字および概念の一覧 → [文字一覧](/characters)
- 今すぐこのブラウザ上でALEV文字を打ち込みたい → [簡易オンラインエディタ](/simple-editor)
- ある程度複雑なALEV文字混じりの文書を作成したい → [Markdownエディタ](/editor)
- PowerPointやExcelなどデスクトップでフォントを使いたい → [インストール](/install)
- 自分のウェブサイトにALEV文字を組み込みたい → [インストール](/install)
- 開発に協力したい、辞書に誤りがある → [GitHub](https://github.com/yukihop/alev1-font)
