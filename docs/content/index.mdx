# ALEV文字解析 & フォント配布プロジェクト

<AlevSignalDemo />

ALEV-1の用いる通称ALEV文字についての解析とフォント配布を行っているサイトです。

## 解析

- ALEV文字と文法の概論 → [ALEV文字・言語について](/intro)
- 実在するALEV文字使用例の確認 → [全文コーパス](/corpus)
- すべての文字および概念の一覧 → [文字一覧](/characters)

## フォントの利用

- 今すぐこのブラウザ上でALEV文字を打ち込みたい → [簡易オンラインエディタ](/simple-editor)
- ある程度複雑な日本語・英語・ALEV文字混じりの文書を作成したい → [Markdownエディタ](/editor)
- PowerPointやExcelなど、デスクトップでフォントを使いたい → [ダウンロード](/install)
- 自分のウェブサイトにALEV文字を組み込みたい → [ダウンロード](/install)
- 開発協力、語彙の拡張、辞書の修正 → [GitHub](https://github.com/yukihop/alev1-font) または作者の X へ
